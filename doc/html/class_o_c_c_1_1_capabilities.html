◆ supportedChecksumTypes()

QList< QByteArray > OCC::Capabilities::supportedChecksumTypes ( ) const

Returns the checksum types the server understands.

When the client uses one of these checksumming algorithms in the OC-Checksum header of a file upload, the server will use it to validate that data was transmitted correctly.

Path: checksums/supportedTypes Default: [] Possible entries: "Adler32", "MD5", "SHA1"
