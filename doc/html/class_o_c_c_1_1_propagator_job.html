the base class of propagator jobs

This can either be a job, or a container for jobs. If it is a composite job, it then inherits from PropagateDirectory

◆ JobParallelism

enum OCC::PropagatorJob::JobParallelism
枚举值
FullParallelism 

Jobs can be run in parallel to this job

WaitForFinished 

No other job shall be started until this one has finished. So this job is guaranteed to finish before any jobs below it are executed.

WaitForFinishedInParentDirectory 

A job with this parallelism will allow later jobs to start and run in parallel as long as they aren't PropagateDirectory jobs. When the first directory job is encountered, no further jobs will be started until this one is finished.
