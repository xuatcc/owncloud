◆ setFileReadOnlyWeak()

void OCC::FileSystem::setFileReadOnlyWeak ( const QString &  filename,
bool  readonly 
)

Marks the file as read-only.

It's like setFileReadOnly(), but weaker: if readonly is false and the user already has write permissions, no change to the permissions is made.

This means that it will preserve explicitly set rw-r–r– permissions even when the umask is 0002. (setFileReadOnly() would adjust to rw-rw-r–)
