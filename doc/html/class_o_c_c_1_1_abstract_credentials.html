virtual void OCC::AbstractCredentials::invalidateToken ( )

Invalidates token used to authorize requests, it will no longer be used.

For http auth, this would be the session cookie.

Note that sensitive data (like the password used to acquire the session cookie) may be retained. See forgetSensitiveData().
