◆ appName()

QString OCC::ownCloudTheme::appName ( ) const
virtual

appName - Application name (short)

Use and redefine this as an application name. Keep it straight as it is used for config files etc. If you need a more sophisticated name in the GUI, redefine appNameGUI.

By default, the name is derived from the APPLICATION_SHORTNAME cmake variable, and should be the same. This method is only reimplementable for legacy reasons.

Warning: Do not modify this value, as many things, e.g. settings depend on it! You most likely want to modify appNameGUI().

返回
QString with app name.
