◆ aboutToRestoreBackup

void OCC::SyncEngine::aboutToRestoreBackup ( bool *  restore)
signal

Emited when the sync engine detects that all the files are changed to dates in the past. This usually happen when a backup was restored on the server from an earlier date. Set *restore to true in a slot connected from this signal to re-upload all files.
